数据绑定-第一个Vue应用

代码：

<!-- 都放在body里即可 --> <div id="app"> <input type="text" v-model="name" placeholder="Your name"> <h1>Hello, {{name}}</h1> </div> <script src="https://unpkg.com/vue/dist/vue.min.js"></script> <script> var app = new Vue({ el: '#app', data: { name: '' } } ) </script>

运行结果：

Hello, {{name}}
